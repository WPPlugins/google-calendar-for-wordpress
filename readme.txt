== Description ==

WPGCalendar is a Wordpress plugin which allows you to integrate Google Calendar in your
Wordpress pages or posts by using the features of the FullCalendar jQuery plugin.

[Check a sample calendar](http://www.customcalendarmaker.com/google-calendar-wordpress/)

== Installation ==

1. Upload the whole calendar directory into your WordPress plugins directory.
2. Activate the plugin on your WordPress plugins page
3. Go to Settings and click WPGCalendar
4. Type in your calendar's XML feed URL (get this from Google Calendar)
5. Type the code [WP-GCALENDAR] on any page or post where you want your calendar to appear.
